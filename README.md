## Install

[npm][]:

```sh
npm install messenger-i18n
```

[yarn][]:

```sh
yarn add messenger-i18n
```


## Usage

```json
{
  "plugins": {
    "i18n" : true
  }
}
```

```json
{
  "plugins": {
    "i18n" : "/path/to/locale/data"
  }
}
```
